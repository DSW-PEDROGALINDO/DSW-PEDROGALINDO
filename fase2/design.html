borrador de contenido para etiquetas semánticas, listas y enlaces

En HTML5 las etiquetas adoptadas sirven para indicar qué on y cuál es el sinificado de los elementos que contienen dentro.
Se denominan Etiquetas Semánticas, las cuales en el marco del HTML y de la componsición de un sitio web son de suma importancia ya que posibilitan a motores de búsqueda como Google a indexar los contenidos de un sitio. Además, dichas etiquetas también proporcionan maquetar páginas web mediante la definición del esquema principal del documento.
Las etiquetas semánticas indican qué es el contenido que enmarcan y cuál es su relación con el conglomerado de elementos del archivo HTML.

Algunas etiquetas semánticas y usos

header: Para definir la cabecera de la página.
footer://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/560

Para definir el pie de página. Normalmente contiene datos sobre quien ha creado la página, datos del copyright, etc.
nav: Permite definir un elemento de navegación de la página web,como el menú principal o menús secundarios.
section: Se utilizan para encerrar el código correspondiente a una sección genérica dentro de un documento.
aside: Sirve para delimitar todo aquel contenido que no es directamente contenido principal de esa página en concreto. Puede usarse por tanto para todos aquellos elementos secundarios como bloques publicitarios, enlaces externos, citas, un calendario, etc.
article: Se utiliza para definir artículos como noticias, entradas de un blog, etc.
